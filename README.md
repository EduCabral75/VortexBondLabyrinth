# Vortex Bond Labyrinth – Nosso Foguete Indie

## Visão Geral
Um puzzle casual mobile onde jogadores tecem laços coloridos em labirintos giratórios, criando portais que ecoam globalmente para conexões anônimas (karma gamificado). Meta: MVP em 12 semanas, lançamento soft na Play Store.
